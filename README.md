# Stack Market Protocol

**Stack Market** is a high-performance decentralized marketplace built on the Stacks blockchain, designed for transparent asset exchange and innovative price discovery. Leveraging the security of Bitcoin, it features automated Dutch Auctions and a premium, institutional-grade interface for the seamless trading of digital assets with real-time on-chain verification.

## 🏗 Project Structure
- **/contracts**: Core Clarity smart contracts for the marketplace and auction logic.
- **/frontend**: Premium React-based dashboard with real-time Stacks blockchain monitoring.
- **/tests**: Comprehensive test suites for contract state and logic verification.
- **/docs**: Technical documentation and protocol specifications.

- **Smart Contracts**: Clarity
- **Frontend**: React, Vite, Tailwind CSS
